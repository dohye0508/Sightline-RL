# ⛰️ Sightline-RL

> **A Reinforcement Learning Environment for Constrained FOV Terrain Exploration**
> (시야각 제한이 있는 지형 탐색 강화학습 시뮬레이터)

**Sightline-RL**은 강화학습(Reinforcement Learning) 에이전트가 장애물과 고저차가 있는 지형에서 '제한된 부채꼴 시야(FOV)'만으로 어떻게 맵 전체를 효율적으로 탐험할 수 있는지 연구하기 위해 만들어진 C++ 환경입니다.

초보자도 쉽게 다운로드하고 실행해볼 수 있도록 친화적으로 구성되어 있습니다. 복잡한 설정 없이 내장된 자동 탐색 알고리즘(Heuristic)이 작동하는 모습을 감상하거나, 키보드로 직접 에이전트를 조종해 볼 수 있습니다!

---

## ✨ 무엇을 하는 프로젝트인가요?

우리가 게임을 하거나 로봇이 미지의 공간을 탐험할 때, **눈앞에 있는 것만 볼 수 있고 산 뒤에 있는 것은 볼 수 없습니다.** (이를 **차폐(Occlusion)**라고 합니다.)
이 시뮬레이터는 이러한 현실적인 제약을 완벽하게 구현했습니다:
- **부채꼴 시야**: 에이전트는 전방 $\pm30^{\circ}$ 범위로 광선(Ray)을 쏘아 지형을 파악합니다.
- **고도의 영향**: 높은 곳(산꼭대기)으로 올라갈수록 장애물 너머를 더 잘 볼 수 있습니다.
- **실시간 안개 걷히기(Fog of War)**: 에이전트가 탐험한 곳은 밝게 빛나며 시각적으로 표시됩니다.

결국, 강화학습 AI가 이 환경과 상호작용하며 **"가장 짧은 시간 안에 맵을 밝히는 최적의 이동 경로"**를 스스로 학습하게 하는 것이 목표입니다.

## 🚀 빠른 시작 가이드 (Quick Start)

### 1. 준비물
가장 빠른 방법은 코드를 직접 빌드하는 것입니다. 윈도우(Windows) 사용자를 기준으로 설명합니다.

- **[CMake](https://cmake.org/download/)**: 프로젝트 빌드 도구 (설치 시 시스템 PATH에 추가 체크 필수!)
- **Visual Studio 2022** (C++ 데스크톱 개발 워크로드 포함)
- **[SFML 2.6.x](https://www.sfml-dev.org/download.php)**: 시각화를 위한 그래픽 라이브러리 (Visual C++ 17 2022 64-bit 다운로드 권장)

### 2. 빌드하는 방법 (Windows)
명령 프롬프트(cmd)나 PowerShell을 열고 아래 명령어를 차례대로 입력하세요.

```bash
# 1. 소스 코드 다운로드
git clone https://github.com/your-username/Sightline-RL.git
cd Sightline-RL

# 2. 빌드 폴더 생성
mkdir build
cd build

# 3. CMake 설정 (SFML 폴더 경로를 본인의 다운로드 경로로 수정해주세요!)
cmake -DCMAKE_BUILD_TYPE=Release -DSFML_DIR="C:/SFML/lib/cmake/SFML" -DUSE_OPENCV=OFF ..

# 4. 컴파일 진행!
cmake --build . --config Release
```

> **💡 팁:** 빌드가 끝나면 `build\bin\Release` 폴더 안에 `rl_constrained_fov.exe` 파일이 생성됩니다.
> 실행하기 전에 SFML 폴더 안의 `bin` 폴더에 있는 모든 `.dll` 파일들을 `.exe`가 있는 곳으로 복사해주세요!

### 3. 시뮬레이터 즐기기
`.exe` 파일을 실행하면 터미널 창에서 두 가지 모드 중 하나를 선택할 수 있습니다:
- `1` **(자동 탐색 모드)**: 내장된 인공지능 규칙에 따라 에이전트가 혼자서 산맥을 오르며 맵을 밝혀나가는 멋진 모습을 감상하세요.
- `2` **(수동 조작 모드)**: 키보드 방향키 `↑`(전진), `←`(좌회전), `→`(우회전)를 눌러 직접 에이전트가 되어 안개를 걷어내 보세요!

## 🧠 강화학습 연구자를 위한 상세 정보

이 시뮬레이터는 Python 환경(Gymnasium 등)과 연동하여 커스텀 강화학습(PPO, SAC 등)을 진행하기에 최적화되어 있습니다. C++의 `RLEnvironment` 클래스를 파이썬 바인딩(Pybind11)을 통해 래핑할 수 있습니다.

- **관측 상태 (State Space)**: 총 66차원 실수 벡터
  - `obs[0~4]`: 에이전트의 현재 X/Y 좌표(정규화), 진행 방향 각도(Sin/Cos), 현재 고도
  - `obs[5~65]`: $-30^{\circ}$ ~ $+30^{\circ}$ 범위의 레이캐스팅 도달 거리 (총 61개 광선)
- **행동 공간 (Action Space)**: 이산형(Discrete) 3개
  - `0`: 전진 (1.0칸 이동)
  - `1`: 좌회전 ($5^{\circ}$ 선회)
  - `2`: 우회전 ($5^{\circ}$ 선회)
- **보상 구조 (Reward Structure)**:
  - ➕ 새롭게 안개를 걷어낸 타일 수에 비례한 큰 탐험 보상
  - ➕ 고도가 높은 곳에 위치할 시 주어지는 실시간 고도(Altitude) 보너스
  - ➖ 아무것도 하지 않고 제자리 회전 시 부여되는 패널티
  - ➖ 이미 밝힌 곳을 맴돌 시 낭비를 막기 위한 재방문 패널티

## 📂 주요 코드 구조 (src 디렉토리)

- **`main.cpp`**: 시뮬레이터 진입점 및 모드 선택
- **`RLEnvironment`**: 강화학습 모델과의 통신을 담당하는 환경 제어기 (`reset()`, `step()`)
- **`Terrain`**: 삼각함수와 가우시안 범프를 조합한 절차적 지형(Heightmap) 생성기
- **`Agent`**: 에이전트의 이동 로직과 물리적 차폐를 고려한 레이캐스팅 연산 담당
- **`Renderer`**: SFML을 이용한 최적화된 2D 시각화 모듈 (높이에 따른 색상 변화 및 Fog of War 구현)
